# Final Project Report

## Project Name

Customer Segmentation & Churn Analytics

## Internship

Unified Mentor Internship Project

## Project Objectives

* Measure overall churn rate
* Analyze churn across customer segments
* Identify high-risk customer groups
* Develop an interactive analytics dashboard

## Work Completed

### Phase 1: Data Cleaning

* Loaded dataset
* Removed unnecessary fields
* Verified data quality

### Phase 2: Feature Engineering

Created:

* AgeGroup
* CreditScoreBand
* TenureGroup
* BalanceSegment

### Phase 3: Exploratory Data Analysis

Analyzed:

* Geography-wise churn
* Age-wise churn
* Gender-wise churn
* Balance-wise churn
* Activity-based churn

### Phase 4: Machine Learning

Implemented Logistic Regression:

* Train-Test Split
* One-Hot Encoding
* Model Training
* Accuracy Evaluation

Results:

* Accuracy: 81.4%

### Phase 5: Dashboard Development

Developed Streamlit dashboard featuring:

* KPI Cards
* Churn Distribution Pie Chart
* Geography Analysis
* Age Analysis
* Tenure Analysis
* Credit Score Analysis
* Gender Analysis
* Active vs Inactive Analysis
* High Value Customer Analysis

## Business Impact

The project provides actionable insights for customer retention strategies and helps identify customer segments requiring targeted intervention.

## Conclusion

The project successfully combines data analytics, customer segmentation, machine learning, and dashboard development to support churn prediction and strategic decision-making.
